# Fleetflow

This is the Fleetflow application containing the frontend and backend.

## Structure
- `frontend/` - Next.js application
- `backend/` - NestJS API
- `.github/workflows/` - CI/CD pipelines
- `docker-compose.yml` - Docker compose file for local development

## Running

1. Start the next.js and nest.js servers
2. Or use docker-compose up
